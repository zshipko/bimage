bimage
======

`bimage` is a basic image library for C.

* Simple API
* 8, 16 and 32 bit images
* Opens/writes 8, 16, 32bit TIFF files
* Opens 8, 16 bit PNG, JPG, PSD, GIF, HDR, BMP files using `stb_image`

## Dependencies

* GNU make
* libtiff
* C99 compiler
* check (for building tests)

## Installation

    make
    make install

To uninstall:

    make uninstall
